1: Co je JavaScript a k jakým účelům se používá?

Doplň vhodné odpovědi a využij k tomu nečíslovaného seznamu.

Použij v textu aspoň 3 internetové odkazy na zajímavé příklady využití JS!

JavaScript je programovací jazyk, který se používá hlavně pro vytváření interaktivních a dynamických prvků na webových stránkách. Umožňuje například reagovat na kliknutí uživatele, měnit obsah stránky bez jejího znovunačtení nebo vytvářet animace. Kromě webových prohlížečů se dnes JavaScript používá i na serverech (například pomocí Node.js) nebo při vývoji mobilních a desktopových aplikací.

3: Jak je možné propojit JavaScript s webovou stránkou?

Doplňte další možnosti ve formě kódu.

Opatřete kód poznámkami!

// 1. Vložit kód do tagu script
<script>console.log('Hello');</script>
// 2. Najde soubor script.js a spustí script v něm
<script src="script.js"></script>
// 3. Kód se vloží přímo do HTML značky, např. u tlačítka:
<button onclick="alert('Hello World');">Klikni na mě</button>
// 4. Načtení jiného souboru přímo z kódu
<script>
let script = document.createElement("script");
script.src = "script.js";
document.body.appendChild(script);
</script>

5: Jaké operátory můžeme používat v JavaScriptu?

Vypište operátory, které patří do uvedených skupin.

U méně obvyklých operátorů dopište do závorky jejich význam!

Matematické

+, -, *, /, % (modulo - zbytek po celočíselném dělení), ** (umocnění)

Přiřazovací

= (přiřazení), += (přičtení a přiřazení), -= (odečtení a přiřazení), *= (násobení a přiřazení), /= (dělení a přiřazení), %= (modulo a přiřazení)

Logické

&& (AND – logický součin), || (OR – logický součet), ! (NOT – negace)

Porovnávací

== (rovnost – porovnává hodnotu), === (přísná rovnost – porovnává hodnotu i typ), != (nerovnost), !== (přísná nerovnost), <, >, <=, >=

Bitové

& (AND – bitový součin), | (OR – bitový součet), ^ (XOR – bitová exkluzivní disjunkce), ~ (NOT – bitová negace), << (posun bitů vlevo), >> (posun bitů vpravo), >>> (posun bitů vpravo bez znaménka)

Typové

typeof (typ proměnné), instanceof (ověření, zda objekt patří určité třídě nebo typu objektu)

6: Která popup okna lze využít v JavaScriptu?

K uvedeným příkladům připište do poznámky, co se stane na webové stránce.

Na konci doplňte aspoň dvě varování, proč bychom se měli vyhýbat použití popup oken v reálných webových aplikacích.

alert()

/* Zobrazí jednoduché informační okno s tlačítkem "OK" */
alert('Nezadal jsi správné heslo!');

prompt()

/* Do proměnné password vloží heslo zadané uživatelem */
<script>let password = prompt('Zadej své heslo:');</script>

confirm()

/* Zobrazí potvrzovací okno. Pokud uživatel klikne "OK", přesměruje ho na stránku seznam.cz. */
if (confirm('Chceš přejít na stránky Seznam.cz?')) location.href='https://seznam.cz';

Varování:
1. Popup okna často ruší uživatelský zážitek a mohou být považována za obtěžující nebo spam.
2. Některé prohlížeče nebo blokátory reklam mohou popup okna automaticky zablokovat, takže nemusí fungovat spolehlivě.
